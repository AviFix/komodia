Komodia's price list.

Required

The Redirector is a core component and is required, it also includes the LSP and Advanced LSP registrar.

Software type

Proxifying solutions

Almost all clients are using just the Redirector.

Parental control/DLP

  • Required: Redirector, HTTP Parser
  • Optional: SSL Digestor, WFP, WD

Ad injection

  • Required: Redirector, HTTP Parser
  • Optional: SSL Digestor, WFP
  • Recommended: WD

Optional components

HTTP Parser

Used to parse HTTP requests and allow easy manipulation, also allows for ad injection.

SSL Digestor

Used to decrypt SSL pages, for example if you want to monitor/modify/inspect HTTPS pages, you need this components.

WFP

Used mostly to intercept Windows Metro traffic (it also intercepts traffic from desktop apps).

Watchdog

  • Is used to protect the SDK against removal by the user.
  • Is used to protect the SDK against removal by the competition (when doing ad injection).

Kernel

Very hard to remove, more aggressive.

User

Easier to remove, less aggressive.
