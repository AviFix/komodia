2014

September/October

  • Released 10th November.

Version 2.3.0.2

  • Fixed a BSOD in 32bit WD when accessing a registry key larger then 1024 bytes.

Version 2.3.0.6

  • Applied the 32bit BSOD fix to the 64bit registry WD.

January-August

Version 2.2.8.16

  • Fixed a bug under Windows 8.1 x64 that protected services would not start.

Version 2.2.8.23

  • Fixed a bug in 64bit file WD to go down cleanly.

2013

July

Version 2.2.6.1

  • Fixes to WD 64bit to better protect processes on Windows 7 and Windows 8.
  • Fixed a problem where a protected non service process couldn't be run with UAC privileges.

January

Version 2.2.3.0

  • WD can't be stopped by using SC stop.

Version 2.2.3.4

  • WD 64bit File, added protection against double uninitialize.

2012

December

Version 2.2.2.8

  • Fixed a situation where the 64bit file WD would crash because of race condition.

May/June

Version 2.1.9.7

  • Some apps ask for full write access to the registry but perform only read, the WD64 will now allow read privileges even if write was requested for protected registry.

Version 2.1.9.3

  • Fixed a problem with WD64 that wouldn't allow to start protect processes that are not services with UAC privileges.

April

Version 2.1.8.4

  • Fixed problems with direct control to the WD.

February

Version 2.1.7.4

Fixes:

  • Fixed the 32bit version to allow perform GetUserName and other user query functions on protected processes.

Version 2.1.7.5

Fixes:

  • Fixed that the 64bit file driver will only inspect FAT and NTFS.

2011

June

Version 2.1.1.7

Fixes:

  • Fixed a condition where adding a process to the protected list would falsely return fail status.

Version 2.1.2.3

Fixes:

  • Fixed a problem that could under certain conditions cause BSOD.
  • Created a specialized version for 32bit Windows 7 that protects a new registry function, now WD distribution comes with two 32bit drivers, one for Windows 7 and one for all the other OS.

Version 2.1.2.6

Fixes:

  • Fixed a bug where a new key could be created under a protected key.

May

Version 2.1.0.4

Release date: 3/5/2011

Fixes:

  • Fixed a bug where a file would be deleted even if protected under 32bit OS.

February

Version 2.1.0.2

Release date: Test version

Fixes:

  • Various BSOD condition fixed.

Version 2.1.0.0

Release date: 2/2/2011

Fixes:

  • Fixed network hang for 32bit and 64bit drivers on Windows 7.
  • Fixed number of BSOD.

2010

December

Version 2.0.0.6

Release date: TBD

Fixes:

  • Changed string functions to safe strings.
  • Fixed a memory leak.
  • Revised the SDK protection rules.
  • Revised the Driver protection rules.

November

Version 2.0.0.4

Release date: 26th November 2010

Fixes:

  • Updated the default rules for 64bit OS.

Version 2.0.0.1

Release date: 11th November 2010

Fixes:

  • Fixed a bug where protected processes would not be intercepted.
  • Vista stability issues fixed.
  • Build server sets correct file version info to Watchdog.

October

Fixed a bug which under certain conditions would not protect values in Windows7 32bit.

September

  • Fixed a bug which caused BSOD on Vista and in general other situations, this affected 32bit and 64bit drivers.
  • Fixed a bug which didn't allow to protect multiple processes under certain conditions.
  • Fixed a bug with 64bit Registry driver that caused it to stop working.

August

Fixed rules to better protect LSP and service under Windows Vista and above.
